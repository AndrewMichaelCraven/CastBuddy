# CastBuddy

CastBuddy is an iPad + Apple TV app that lets a parent or caregiver pick what plays on the TV from a simple, picture-based library — no remote required for the person watching.

Built with kids in mind, especially those who struggle with the Siri Remote — including non-verbal children, kids with autism, and anyone for whom navigating menus is a barrier. CastBuddy turns the parent's iPad into a focused, picture-based launcher that hands content directly to the TV the child is already watching.

## How It Works

1. Install **CastBuddy** on the iPad and **CastBuddy** on the Apple TV (both are bundled in the same App Store listing)
2. Pair them with a 4-digit code (one-time setup)
3. Open Child Mode on the iPad and tap a picture
4. The matching video opens on the Apple TV
